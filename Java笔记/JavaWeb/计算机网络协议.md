## URL编码

URL编码是将 URL中不可打印的字符或具有特殊含义的字符转换为Web浏览器和服务器普遍接受的字符的过程。

URL中的汉字都要进行URL编码。(把汉字字符串转成字节再转成URL字符串。浏览器会对URL编码进行解码并读取信息)。

原本的一个字符会被编码成 一个百分号和两个十六进制数字
