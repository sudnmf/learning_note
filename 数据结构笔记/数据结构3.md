## 7.2 顺序搜索

### 无序表的顺序搜索

![image-20221019090046809](数据结构3.assets/image-20221019090046809-16661412651811.png)

```
int SeqSearch(ListSet L, ElemType x)
{ 
	int i;
	for (i=0; i<L.n; i++)
		if (L.element[i]==x) {
			return i; //搜索成功
		}
	return -1; //搜索失败
}
```
